picgo + github 搭建图床

1. github 搭建图床

1.1 新建图床仓库

github账号 使用博客的即可

  • 打开 GitHub ，点右上角 + ，弹出菜单点击 New Repository

  • 填写 Repository Name 为 imagebed
  • 设置为公开 Public
  • 点击 Create repository 创建仓库

1.2 生成令牌 (token)

  • 进入 token设置页面 ，点击 Generate new token
  • 填写令牌描述为 imagebed
  • 设置里勾选 repo

  • 拉到网页底部，点击 Generate token 生成令牌

※务必复制保存这串令牌备用，GitHub只会显示这一次

2.Picgo 多图床上传客户端

2.1 Picgo 稳定版下载并安装

Picgo 2.2.2

默认安装即可，Picgo 基础设置如图

2.2 github 图床设置

  • 设定仓库名
  • 设定分支名
  • 设定 token
  • 指定存储路径
  • 设定自定义域名 (这里套用cdn加速)

2.3 pic-migrater 图床批量迁移插件

  • 进入 插件设置
  • 搜索 pic-migrater 并安装
  • 显示 已安装 表示安装完成，重启 Picgo

  • 重启后进入 插件设置
  • 点击 齿轮 图标 ，点选 配置plugin - pic-migrater

  • 文件名后缀键入 _new 后 确定 完成插件使用前配置

点击 齿轮 图标 ，点选 选择文件 开始使用

该插件仅支持Markdown，图片过多有迁移失败可能
